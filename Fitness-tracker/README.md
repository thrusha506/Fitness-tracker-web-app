BalanceBeat Fitness Tracker Web Application



Theme

The theme of this web application is a "Fitness Tracker".  
The system helps users track their daily fitness activities such as steps taken, calories burned, and exercise duration. It also helps users monitor their progress and achieve their fitness goals.


Features

Log daily fitness activities
Track steps taken and calories burned
View activity history
Set daily and weekly fitness goals
Contact form for user communication
Responsive design using Bootstrap


Usage Instructions

1. Open the project folder.
2. Open the "home.html" file in a web browser.
3. Navigate through the pages using the navigation bar.
4. Go to the "Activity page" to log daily activities.
5. View your progress in the "Progress page".
6. Set fitness targets in the "Goals page".
7. Use the "Contact page" to send messages or feedback.
